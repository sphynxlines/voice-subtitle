📖 使用帮助

🎤如何使用

  1. 点击绿色的「开始」按钮
  2. 第一次使用会询问麦克风权限，点击「允许」
  3. 对着手机说话，文字会显示在屏幕上
  4. 点击红色的「停止」按钮结束

📱添加到主屏幕

添加后可以像普通 App 一样使用

  1. 用 Chrome 浏览器打开本页面
  2. 点击右上角菜单 ⋮
  3. 选择「添加到主屏幕」或「安装应用」
  4. 点击「添加」
  5. 桌面会出现「语音字幕」图标
  1. 用 Safari 浏览器打开（不是 Chrome）
  2. 点击底部的分享按钮 ⬆️
  3. 向上滑动，找到「添加到主屏幕」
  4. 点击右上角「添加」
  5. 桌面会出现「语音字幕」图标
⚠️ 苹果手机必须用 Safari 浏览器，用 Chrome 或微信打开无法添加

🔐麦克风权限

如果不小心拒绝了权限，可以这样恢复：

  1. 打开手机「设置」
  2. 找到「应用管理」或「应用」
  3. 找到「Chrome」或「语音字幕」
  4. 点击「权限」
  5. 找到「麦克风」，选择「允许」
  1. 打开手机「设置」
  2. 向下滑动找到「Safari」
  3. 点击「麦克风」
  4. 选择「允许」

❓常见问题

没有声音 / 不显示文字？
请检查是否允许了麦克风权限，并确保网络连接正常。
识别不准确？
尽量在安静的环境使用，说话清晰一些，手机放近一点。
屏幕会变暗？
正在录音时屏幕会保持常亮。如果还是变暗，可以在手机设置里延长屏幕自动锁定时间。
显示「网络已断开」？
请检查手机是否连接了 WiFi 或移动数据。
